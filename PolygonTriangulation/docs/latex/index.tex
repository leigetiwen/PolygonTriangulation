{\bfseries{Converting to y-\/monotone\+:}} Constructing the priority queue takes O(nlogn) for n vertices as each is handled after sorting. ~\newline
 Queries and updates on binary trees take O(logn) time. Thus handling all the events takes O(nlogn) time.~\newline
 The algorithm takes O(n) space.~\newline
 {\bfseries{Triangulation}} Since, the number of pushes is bounded by O(n), the number of pops from the stack cannot be greater than ~\newline
 that. Thus, the time complexity is O(n) ans space complexity is also O(n).
